- Added `test-off-canvas-panel-a11y.mjs` and a required Production CI step.
- All 27 deterministic Maps regression scripts passed.
- `npm ci`, `npm run build`, and `git diff --check` passed.
- Local browser verification confirmed open state removes `inert` and sets `aria-hidden="false"`; close restores `inert`, `aria-hidden="true"`, and focus to `openDataManagerBtn`.
- Expected local-only MapKit origin errors occurred because the checked-in token permits `lalgeo.com`, not `127.0.0.1`; production MapKit loaded successfully.
